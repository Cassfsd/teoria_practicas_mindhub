La estructura del grid tiene que estar definidas por ambas, filas y columnas, si solo coloco una o la otra no va a mostrar la estructura. Grid con fracciones: se modifican de acuerdo al tamaño del contendor padre. Podemos usar px pero es un valor fijo.

-La función minmax(), se utiliza para definir rangos flexibles de celda. -Función repeat, permite mediante auto-fill y auto-fit indicarle al navegador que rellene o ajuste el contenedor grid con múltiples elementos hijos, dependiendo del tamaño del viewport.

Ejemplo: repeat(auto-fill,minmax(300px, 1fr)). "Se van a repetir las filas Se encarga de que los elementos hijos con el tamaño mínimo queden en la primera fila y los que no caben, se desplazan a la siguiente fila del grid.
