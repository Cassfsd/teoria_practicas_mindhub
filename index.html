Justifiy Content

  1. flex Start
    2. Alinea el contenido a la izquierda
  2. flex end
    3. Alineamos a la derecha
  3. center
    4. alinea horizontalmente al centro
  4. space-between
    5. alinea de forma uniforme los elementos con espacio entre cada uno de ellos, pero no entre el contenedor y el borde del elemento
  5. space-around
    6. alinear a lo largo del contenedor, ahora si teniendo espacio entre el elemento y el borde del contenedor
  6. space-evenly
    7. ditribuye los elementos a la misma distancia entre ellos y entre el contenedor y sus bordes

    Notas: Flexbox se aplica a los hijos directos del contenedor, por lo que flex deberemos aplicarlo a cada uno de los contenedores en donde queramos que su contenido hijo se vea afectado por el layout de flex.
